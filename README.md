# Кресты — Изоляция приложений

Приложение для изоляции других Android-приложений с использованием встроенного рабочего профиля Android. Не требует root-прав.

## Особенности

- **Без root-прав** — использует стандартный API Android Work Profile
- **Простой интерфейс** — минималистичный дизайн Material You
- **Заморозка приложений** — временное отключение приложений без удаления
- **Изоляция данных** — приложения в рабочем профиле не имеют доступа к личным данным
- **Монетизация через RuStore** — безопасные платежи для российских пользователей

## Требования

- Android 10 (API 29) или выше
- Поддержка рабочих профилей на устройстве
- Установленный RuStore для оформления подписок

## Как это работает

Приложение использует функцию Android "Work Profile" (Рабочий профиль), которая изначально предназначена для корпоративного использования. Work Profile создает отдельное изолированное пространство внутри одного пользователя.

### Преимущества:
- Приложения в Work Profile не имеют доступа к данным основного профиля
- Можно "замораживать" приложения — они исчезают из списка и не работают в фоне
- Удаление рабочего профиля удаляет все данные изолированных приложений

## Техническая информация

### Используемые API:
- `DevicePolicyManager` — управление рабочим профилем
- `ACTION_PROVISION_MANAGED_PROFILE` — создание профиля
- RuStore Billing SDK — монетизация

### Архитектура:
- Kotlin
- Material Design 3 компоненты
- RuStore Pay SDK 10.0.0
